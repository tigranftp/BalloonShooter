BalloonFighters

Introduction
Demo game for Simple 3D Tilemap Editor
This game is acade game to attack the balloon above the head of the enemy
if used Simple 3D Tilemap Editor you can add new map

How to Play(Keys)
Move : Left Arrow, Right Arrow
Fly     : Space
